PolyGuru

Summary

Polyguru is a productivity platform designed to empower students through Generative AI tools. It offers features like test paper generation using Gemini AI, a comprehensive GPA calculator, and a collaborative Q&A feed to enhance learning efficiency.

My Role

  • Served as a full-stack developer, working alongside a teammate.
  • Designed and implemented the Test Paper Generation tool using Gemini AI.
  • Developed the GPA Calculator and the community Q&A feed.
  • Handled database access with Prisma ORM and integrated AI components.
  • Created a responsive UI with TailwindCSS and DaisyUI.

Visuals

Test Paper Generation using Gemini AI

Developed an AI-powered tool that generates customized test papers for students based on their learning materials. Integrated with Google's Gemini AI for intelligent question creation and built a user-friendly interface for test customization.

GPA Calculator

Developed an intuitive GPA calculator tailored for polytechnic students. The tool allows users to input their grades and credit units to calculate both semester and cumulative GPAs, with an option to save and track progress over time.

Work Process

PolyGuru was born out of a shared goal between my teammates and me to improve how students prepare for exams and track their academic progress. We identified a need for tools that could automate test creation and simplify GPA management.

We worked together on the full-stack development, focusing on core features like AI-powered test paper generation and an intuitive GPA calculator, alongside a collaborative Q&A feed.

Outcome & Results

PolyGuru is now a functional platform that empowers students to create custom test papers from their notes and accurately calculate their GPA. Initial feedback has been positive, with users highlighting the test generator's effectiveness for revision and the GPA calculator's convenience.
